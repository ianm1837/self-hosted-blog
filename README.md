## Installation

### Docker Compose
Requirements:
- Docker Engine
- Docker Compose

Steps:
- Clone Repo
- Copy .env.EXAMPLE to .env
- Fill out environment variables
- create ```db_data``` folder in root of app
- run: ```docker compose up -d```

### Locally

Requirements:
- node js
- MySQL Server

Steps:
- Clone Repo
- Copy .env.EXAMPLE to .env
- Fill out environment variables
- run: ```npm install```
- run: ```npm run start```
